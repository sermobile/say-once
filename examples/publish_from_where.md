# «Выложи лабу» — откуда?

## Ситуация
Александр просит Сергея опубликовать последнюю лабу на сервер, чтобы можно было пиарить релиз.

## Как дали информацию
> «Выкинешь самую последнюю лабу на set, чтобы уже можно было отправлять ссылку»

Сергей: «Сегодня выложу» — и через 1.5 часа:

> «эммммммм, мы из девел выкладывать будем?»

## Что было не так

**Два нарушения одновременно:**

1. **Александр** не указал ключевой параметр задачи — откуда публиковать (ветка/окружение). Задача была поставлена с пропуском критичной детали.

2. **Сергей** подтвердил задачу («сегодня выложу»), не имея всей информации для её выполнения. Создал ложное ощущение, что всё понятно. Уточнил только в момент, когда сел делать — спустя 1.5 часа.

## Как надо было

**Александр** при постановке задачи:
> «Выкинешь последнюю лабу на set из ветки X?»

**Сергей** при получении неполной задачи — сразу:
> «Выложу сегодня. Из девела берём?»

## Принципы
- Задача должна содержать все параметры, необходимые для её выполнения — без додумывания.
- Подтверждение задачи без уточнения недостающих деталей — это ложное подтверждение.
