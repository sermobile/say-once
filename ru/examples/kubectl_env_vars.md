# Переменные окружения в Kubernetes

## Ситуация
В общем чате спросили, как реализовать обходной путь (workaround).

## Как дали информацию
Сотрудник скинул просто список переменных окружения с нужными значениями:

```
FOO=bar
BAZ=qux
```

## Что было не так
Тому, кто спрашивал, пришлось самому:
- понять, что это переменные окружения для Kubernetes
- вспомнить или найти нужную команду (`kubectl set env ...`)
- корректно сформировать команду с этими переменными
- не ошибиться в синтаксисе

## Как надо было
Дать сразу готовую команду:

```bash
kubectl set env deployment/my-deployment FOO=bar BAZ=qux
```

## Принцип
Информация должна быть в том виде, в котором получатель может её **немедленно использовать** — без промежуточных преобразований.
