# say-once

> Книга о том, как сказать один раз — и быть понятым.
> Say it once. Be understood.

```
                  ┌──────────────────────────┐
                  │         ПРАВИЛА          │
                  │          RULES           │
                  └─────────────┬────────────┘
                                │
       ┌────────────────────────┼────────────────────────┐
       │                        │                        │
       ▼                        ▼                        ▼

   ЧЕЛОВЕК ↔ ЧЕЛОВЕК       ЧЕЛОВЕК ↔ UI         ЧЕЛОВЕК ↔ АГЕНТ
   HUMAN ↔ HUMAN           HUMAN ↔ UI           HUMAN ↔ AGENT
```

Правила одни и те же — для людей, интерфейсов и ИИ-агентов. Меняется только цена ошибки.
The rules are the same — across people, interfaces, and AI agents. Only the cost of breaking them changes.
